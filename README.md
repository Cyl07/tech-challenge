# Tech Challenge

## Install

In order to install the project, follow the step bellow

1. Clone the repository:
```bash
git clone git@github.com:Cyl07/tech-challenge.git
cd tech-challenge/
```

2. Import the database:
```bash
mysql -u <pseudo> -p < database.sql
```

3. Run the server:
```bash
php -S localhost:8000
```
